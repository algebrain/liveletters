Платформонезависимый клиент IMAP, как для PC (в первую очередь) так и для мобильных (потом). По интерфейсу livejournal

Технология для PC: Tauri

Фичи:

- посты
- дружбы
- подписки
- бан
- подзамок
- только для друзей
- сообщества с владельцем и модераторами
- сообщества только для участников
- лайки
- древовидные обсуждения

Ну и т.д. по минимуму, что входит в привычки в ЖЖ

Письма пересылаются в открытую, без шифрования
Каждое техническое сообщение пересылается вместе с информативным сообщением юзеру
Техническая часть сообщения скрыта, но при желании юзер может ее посмотреть
Можно часть техники пересылать в специальных заголовках письма (x-...)
